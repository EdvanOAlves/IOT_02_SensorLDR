# IOT_02_SensorLDR

Atividade desenvolvida com ESP32 como prática do componente detector de luminosidade, o circuito liga o led ao receber o sinal de baixa luminosidade do sensor LDR. Sesenvolvido para a disciplina de IOT do curso Desenvolvimento de Sistemas do Senai

Segue um vídeo do código sendo executado em circuito:
https://www.youtube.com/watch?v=Wau44QjFRl8

## Tecnologias utilizadas
* C++
* Arduino IDE
* Wokwi
* ESP32
